## 시작은 간단했다

한국 주식 종목 하나 찍으면 기본 정보 + 투자자별 매매동향 + 기술적 지표를 싹 모아서 **텔레그램으로 쏴주는 봇**이 있으면 좋겠다는 생각. 네이버 금융·WiseReport·pykrx·한국은행 ECOS API를 스크래핑·조합하고, 자연어 질문은 `claude -p`에 컨텍스트 주입해서 대답하는 방식.

여기까지는 주말 두어 번이면 됐다.

## 욕심이 생겼다 — ML 앙상블

"단순 지표 나열보다 **상승 확률** 같은 거 찍어주면 더 있어 보이지 않나?"

- **데이터**: KOSPI 30종목 × 5년, 33,128건
- **라벨**: 20 거래일 후 종가가 ±5% 움직였는지 (상승/하락/보합)
- **피처 15개**: SMA 비율, RSI, MACD, 볼린저밴드 위치, 스토캐스틱, 거래량 변화율, 변동성, 수익률 구간 등
- **모델**: XGBoost 분류기
- **백테스트**: 상승 예측 적중률 62.5%, 평균 수익률 +5.81%
- **최적 가중치**: ML 75% + 규칙 기반 25% 앙상블

자연어 응답 첫 줄에 "매수/매도 점수 X점 (ML Y% / 규칙 Z%)" 이 박히니 봇이 갑자기 **그럴듯해 보였다.**

## 강화학습까지 뻗어봤다

분류 문제를 풀 게 아니라 **실제 매매 정책(policy)을 직접 배우게 하면 어떨까**. 상태: 현재 포지션 + 가격 윈도우 + 지표, 액션: {매수, 보유, 매도}, 보상: 수익률. DQN 기반으로 굴려봤는데 백테스트 그래프가 꽤 예쁘게 올라갔다 (물론 실전은 별개의 문제지만).

## 웹 플랫폼도 올렸다

봇 혼자서는 공유하기 애매하니 **웹으로 래핑**:
- Flask + Plotly로 대시보드 (거시경제 카드 + 외국인/기관 순매수 TOP 10)
- 종목 페이지: 인터랙티브 차트 + 앙상블 점수 + 시그널 + 증권사 목표가
- Nginx + Let's Encrypt, 다크 테마
- 라즈베리파이에 `stock-advisor-web` systemd로 올림 → **https://pullystock.duckdns.org**
- `/investor`, `/fundamental`, `/technical`, `/macro`, `/score` API 엔드포인트

## 문득 — 이걸 사업으로 미는 곳이 이미 있을 것 같았다

웹까지 올리고 대시보드가 어느 정도 모양 나니까, 이렇게 딸깍으로 만든 걸 누군가 이미 사업 아이템으로 밀고 있을 것 같다는 생각이 들어서 검색해봤다.

있었다. **핀트(Fint).** 로보어드바이저 서비스. ML 기반 자동 매매, 성향별 포트폴리오, 앱까지 깔끔하게. 심지어 실제 계좌 연동으로 자동 매매까지 돌려준다. 내가 "다음 단계로 뭐 해야 되지" 하던 기능들이 **이미 다 구현돼 있었다.**

결국 나도 거기로 넘어갔다 ㅋㅋ. 실제 돈 넣을 땐 **내가 백테스트 돌린 모델보다 검증된 서비스가 낫다**는 걸 인정해야 했다.

## 그래도 남은 것

이 프로젝트에서 포기한 건 아니다. 오히려 **가장 값진 경험**이 됐다:

- **엔드투엔드 파이프라인 경험**: 데이터 수집(스크래핑·API) → 피처 엔지니어링 → 학습 → 백테스트 → 서빙(봇·웹) → 배포(파이 + systemd + HTTPS). 한 사람이 한 사이클 다 돌려본 건 처음
- **`claude -p`로 자연어 레이어 만들기**: 데이터 컨텍스트를 프롬프트에 잘 주입하면 봇이 그럴듯한 애널리스트처럼 답함. API 추가 비용 없이
- **"시장에 이미 있다"의 빠른 확인이 중요하다**: 더 깊이 파기 전에 검색 한 번으로 내 몇 주짜리 플랜이 절약됨
- **개인 프로젝트로는 만들 수 있다. 서비스로는 자본·규제·신뢰가 다른 게임이다**

## 지금 상태

- 라즈베리파이에서 여전히 **돌고는 있다** (`stock-advisor-bot` / `stock-advisor-web` 둘 다 active)
- 실제 포트폴리오는 핀트에 맡김
- 강화학습 쪽은 "실전 투자에 안 쓰는 장난감" 포지션으로 유지 — 나중에 딴 데 응용할 여지
- 웹은 디자인이 투박해서 손볼 것들이 있는데, 투자 대비 효용을 생각하면 우선순위가 낮아짐
